Exercise 3 - Responsive Grid Layout

Column 1

On mobile, this appears as a single column per row; on tablets, it shows two side by side, and on desktops, three columns are displayed per row.

Column 2

This displays one column per row on phones, two columns per row on medium-sized devices, and three columns across larger screens.

Column 3

Mobile devices show each column separately; tablets show two in a row, while desktops arrange three in one row.
